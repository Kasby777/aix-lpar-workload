This plugin uses the "lslparutil" command on the HMC to gather the utilization metrics.  For more information, please see:

http://pic.dhe.ibm.com/infocenter/director/v6r2x/index.jsp?topic=%2Fdpsm%2Fdpsm_reference%2Fmanpages%2Fpsm.lslparutil.html

On a POSIX Monitoring Station, keyless SSH needs to be configured connected from the Monitoring Station to the HMC.  The following are metrics gathered by the plugin:

Chassis CPU Utilization (%)
Chassis CPU Utilization (CPU unit)
LPAR CPU Utilization (%)
LPAR CPU Utilization (CPU unit)
Response Time (ms)
